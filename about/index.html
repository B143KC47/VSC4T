👋 嗨，我是 B143KC47 | Hi, I’m B143KC47

你好！我是 B143KC47，VSC4T Hexo 主题的创建者和主要维护者。作为一名热爱编程、设计和开源分享的技术爱好者，我始终相信优雅的设计和高效的功能可以和谐共存。

我开发 VSC4T 的初衷源于对 Visual Studio Code 编辑器界面美学的欣赏，以及希望为 Hexo 用户带来一种新颖、熟悉且高效的博客体验。我希望将代码编辑器的简洁、专注和开发者友好的特性融入到博客的写作与阅读之中。

Hello! I’m B143KC47, the creator and main maintainer of the VSC4T Hexo theme. As a tech enthusiast passionate about programming, design, and open source sharing, I firmly believe that elegant design and efficient functionality can coexist harmoniously.

My motivation for developing VSC4T stems from my appreciation for the aesthetics of the Visual Studio Code editor interface, and my desire to bring Hexo users a fresh, familiar, and efficient blogging experience. I aim to incorporate the simplicity, focus, and developer-friendly features of code editors into the writing and reading experience of blogs.

✨ 关于 VSC4T 主题 | About VSC4T Theme

VSC4T 是一款受 VS Code 启发的、优雅的暗色 Hexo 主题。它不仅仅是 VS Code 界面的简单模仿，而是将编辑器的设计理念与博客平台的需求相结合，创造出一种独特的阅读与写作体验。

VSC4T 的核心设计理念是：

为技术内容创作者和读者提供一个沉浸式、代码友好的环境，让写作和阅读技术博客变得自然而高效。

VSC4T is an elegant dark Hexo theme inspired by VS Code. It’s not merely a simple imitation of the VS Code interface, but a fusion of editor design philosophy with blogging platform requirements, creating a unique reading and writing experience.

The core design philosophy of VSC4T is:

To provide an immersive, code-friendly environment for technical content creators and readers, making the writing and reading of technical blogs natural and efficient.

🎨 设计理念 | Design Philosophy

  • 熟悉感：对于开发者来说，VSC4T 提供了类似日常编码环境的界面，降低了上下文切换的认知负担

  • 可读性优先：精心选择的对比度、字体大小和行间距，确保长时间阅读的舒适度

  • 功能性美学：每个设计元素都有其功能目的，避免纯粹的装饰性干扰

  • 一致性体验：从颜色方案到交互模式，保持整体界面的一致性

  • Familiarity: For developers, VSC4T provides an interface similar to their daily coding environment, reducing the cognitive load of context switching

  • Readability First: Carefully selected contrast, font sizes, and line spacing ensure comfort during long reading sessions

  • Functional Aesthetics: Each design element has its functional purpose, avoiding purely decorative distractions

  • Consistent Experience: From color schemes to interaction patterns, maintaining consistency throughout the interface

🚀 核心特性 | Core Features

VSC4T 包含了一系列旨在提升博客体验的功能：

  • 🎨 VS Code 界面风格：熟悉的暗色调、图标和布局
  • 📱 响应式设计：在桌面和移动设备上均提供良好体验
  • 🧭 类资源管理器侧边栏：直观地浏览分类和标签
  • 💻 优化的代码高亮：提供类似 VS Code 的代码阅读体验，支持行号、代码复制等
  • 🔍 内置搜索功能：快速查找文章标题、内容或标签
  • 🌐 多语言支持：已支持包括中文在内的12种语言
  • 📊 Mermaid 图表：直接在 Markdown 中绘制流程图、序列图等
  • 🔧 易于配置：提供清晰的配置选项
  • 🌙 深度优化的暗色模式：精心调整的颜色方案，减轻眼睛疲劳
  • ⚡ 性能优化：快速加载和渲染速度

VSC4T includes a series of features designed to enhance the blogging experience:

  • 🎨 VS Code Interface Style: Familiar dark tones, icons, and layout
  • 📱 Responsive Design: Provides excellent experience on both desktop and mobile devices
  • 🧭 Explorer-like Sidebar: Intuitive navigation of categories and tags
  • 💻 Optimized Code Highlighting: Provides VS Code-like code reading experience with line numbers, code copying, etc.
  • 🔍 Built-in Search Function: Quickly find article titles, content, or tags
  • 🌐 Multi-language Support: Already supports 12 languages including English and Chinese
  • 📊 Mermaid Diagrams: Draw flowcharts, sequence diagrams, etc. directly in Markdown
  • 🔧 Easy Configuration: Provides clear configuration options
  • 🌙 Deeply Optimized Dark Mode: Carefully adjusted color scheme to reduce eye strain
  • ⚡ Performance Optimization: Fast loading and rendering speed

  • 🌍 Translate: Help translate the theme into more languages
