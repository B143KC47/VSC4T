Waline Setup | Waline 设置

Your theme already integrates the Waline comment system. You only need to deploy the server to start using it.

您的主题已集成 Waline 评论系统，只需部署服务端即可使用。

Quick Start (≈5 minutes) | 快速部署（约 5 分钟）

Option A: Deploy on Vercel (Recommended) | 方式一：Vercel 部署（推荐）

  1. One‑click deploy | 一键部署：

    • Deploy to Vercel
    • Log in with GitHub and click “Create” | 使用 GitHub 登录并点击 “Create”

  2. Get your server URL | 获取服务器地址：

    • After deploy, click “Visit” to open your Waline URL | 部署成功后点击 “Visit” 打开 Waline 地址
    • Format | 地址格式：https://your-project-name.vercel.app

  3. Update theme config | 更新主题配置：
    Edit themes/VSC4T/_config.yml and set your server URL | 编辑 themes/VSC4T/_config.yml 并设置服务器地址：

Option B: LeanCloud + Vercel (Safer data) | 方式二：LeanCloud + Vercel（数据更安全）

  1. Register LeanCloud (International) | 注册 LeanCloud 国际版：

    • Visit the console | 访问控制台：LeanCloud Console
    • Create an app (Developer plan, free) | 创建应用（选择开发版，免费）
    • Go to App > Settings > Credentials and note | 进入 应用 > 设置 > 应用凭证 并记录：App ID、App Key、Master Key

  2. Deploy to Vercel | 部署到 Vercel：

    • Use the one‑click deploy link above | 使用上面的一键部署链接
    • Add environment variables | 添加环境变量：
  1. Initialize data tables | 创建数据表：
    • Open: https://your-project-name.vercel.app/ui
    • Sign in with the environment values | 使用环境变量中的信息登录
    • The system will auto‑create required tables | 系统会自动创建所需数据表

Custom Domain (Optional, Recommended) | 绑定自定义域名（可选但推荐）

Because vercel.app may be unstable in some regions, bind a custom domain.

由于 vercel.app 域名在部分地区可能不稳定，建议绑定自定义域名。

  1. In Vercel project settings, open “Domains” | 在 Vercel 项目设置中点击 “Domains”
  2. Add your domain (e.g., waline.yourdomain.com) | 添加你的域名（如：waline.yourdomain.com）
  3. Follow the instructions to add a CNAME record | 按提示在 DNS 中添加 CNAME 记录

Function Test | 功能测试

  1. Visit admin UI | 访问管理后台：https://your-waline-server.vercel.app/ui
  2. Test leaving a comment on a post | 在博客页面测试评论
  3. Verify email/Telegram notifications | 检查邮件/Telegram 通知是否正常

FAQ | 常见问题

  • Q: Comment box doesn’t show? | 评论框不显示？

    • A: Check serverURL in theme config and browser console errors | 检查主题配置中的 serverURL 及浏览器控制台是否有错误

  • Q: How to migrate from other systems? | 如何迁移其他评论系统的数据？

    • A: Waline can import from Valine, Disqus, Twikoo via admin UI | Waline 支持从 Valine、Disqus、Twikoo 等导入，在管理后台操作

  • Q: Customize comment styles? | 如何自定义评论框样式？

    • A: The theme already adapts VS Code style; tweak in source/css/main.css or site CSS | 主题已适配 VS Code 风格，可在 source/css/main.css 或站点样式中调整
